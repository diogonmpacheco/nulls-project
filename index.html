Interpretation frame

A null is not one thing.

Each record separates mechanism from meaning, so a user can see whether the story is inherited absence, residual activity, temporary suppression, or protective loss of function.

Inherited null

A no-function allele, deletion, or missing function that may affect every compartment where the gene is normally active.

Low-function review

Severe reduction that can approximate a null in some contexts, but should not be collapsed into a true absence.

Induced functional null

A drug or exposure can temporarily suppress function, but distribution and timing decide which tissues are actually silenced.

Protective LoF

Some missing-function states are protective, showing why less function is not automatically pathological.
